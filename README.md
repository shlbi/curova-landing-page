# Curova Landing Page

<img width="800" alt="screenshot" src="https://github.com/user-attachments/assets/86fc2127-c850-43b0-a227-269f3e716472" />

Static landing page for the Curova project. This repository contains a simple HTML landing page (`index.html`).

## View locally

- Open `index.html` in your browser.
- Or serve locally with Python's simple HTTP server:

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000 in your browser.

## Development

- Edit `index.html` and assets directly.
- Commit changes and push to the repository.
